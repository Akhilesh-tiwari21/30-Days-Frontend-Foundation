## 🎯 Age Calculator App

# Introduction
A clean and interactive Age Calculator built using HTML, CSS, and Vanilla JavaScript.
This app calculates a user’s exact age in years, months, and days and also shows a live countdown to the next birthday.

## 🚀 Features
* 📅 Select Date of Birth using date picker
* 🔢 Accurate age calculation (Years, Months, Days)
* 🎂 Detects today’s birthday and shows a special message
* ⏳ Live next birthday countdown (days, hours, minutes, seconds)
* 🚫 Prevents future date selection
* ✨ Smooth number animation for better UX
* 📱 Responsive and clean UI
* ⚡ Built with pure JavaScript (no libraries)

## 🧠 Logic Highlights
* Handles month & day adjustment correctly
* Prevents negative values in age calculation
* Uses Date object efficiently
* Separates logic into small reusable functions
* Handles edge cases like:
* Empty input
* Future dates
* Birthday today

## 🛠️ Tech Stack
* HTML5 – Semantic structure
* CSS3 – Modern styling & animations
* JavaScript (ES6) – Core logic & DOM manipulation

## ▶️ How to Run

* Clone the repository
``` git clone <repo-url> ```

* Open ageCalculator.html in your browser
* Select your date of birth
* Click Calculate Age
