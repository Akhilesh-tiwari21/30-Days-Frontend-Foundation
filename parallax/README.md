## 🌌 Parallax Portfolio

# Introduction
A simple and visually appealing Parallax Portfolio Website built using HTML and CSS.
This project demonstrates how to create fixed background scrolling effects and structured content sections for a personal portfolio.

## 📖 About The Project

* This website includes:

* A parallax-style header section
* An introduction section
* A projects showcase section
* Fixed background image effect
* Clean and minimal layout structure

## It is a beginner-friendly project focused on understanding layout, spacing, and background effects using pure CSS.

## 🚀 Technologies Used
* HTML5
* CSS3
* Background Parallax Effect (background-attachment: fixed)
* Basic Box Model
* Typography Styling

## 🎯 Key Concepts Used
* 1️⃣ Parallax Effect
The background is fixed while content scrolls over it using:

* background-attachment: fixed;
* background-size: cover;

* 2️⃣ Section Styling

* .head-container → Used for large parallax header section
* .para-1 → About section
* .para-2 → Projects section
*  Each section is styled with padding, background color, and text formatting.
