## 📝 Survey / Feedback Form (Frontend Task)

# Introduction
A clean and beginner‑friendly **Survey Form** built using **HTML and CSS**.
This project focuses on understanding **form elements**, **input types**, and **basic form styling** while maintaining clean structure and readability.

## 🎯 Project Objective

The main goals of this task are:

* Learn how HTML **forms** work
* Practice different **input types** (text, email, radio, checkbox, textarea)
* Understand **label–input association**
* Style forms using **basic CSS**
* Build confidence in real‑world frontend form layouts

This project is part of my **Frontend Foundations / Daily Practice** journey.

## ✨ Features

* 📄 Structured survey form layout
* 🧾 Text, email, and number inputs
* 🔘 Radio buttons for single choice
* ☑️ Checkboxes for multiple selections
* 📝 Textarea for feedback and comments
* 🎨 Clean and simple UI using CSS
* 📱 Responsive and centered form

## 🛠️ Tech Stack Used

* **HTML5** – Form structure and semantics
* **CSS3** – Layout, spacing, and visual styling


## ⚙️ How the Form Works

### 🔹 HTML

* `<form>` wraps all input elements
* `<label>` improves accessibility and UX
* `<input>` fields handle user data
* `<select>` allows dropdown selection
* `<textarea>` captures user feedback

Each input is grouped inside `.form-control` for clean layout control.

### 🔹 CSS

* Page background styled for contrast
* Form is centered using margin auto
* Inputs are full‑width for better usability
* Radio buttons and checkboxes styled inline
* Button styled to match theme color

## 🧠 Key Learnings

* Practical use of HTML form elements
* Importance of labels for accessibility
* Grouping form fields logically
* Basic but effective CSS form styling
* Writing readable and maintainable code
