# 🎨 Color Generator App

##  Introduction
A simple and interactive Color Generator built using HTML, CSS, and JavaScript.
This project helps in understanding DOM manipulation, event handling, and basic UI/UX logic.

Designed with a creative + corporate frontend approach.

## 🚀 Features
1. 🎲 Generate random colors on button click
2. 🎯 Same Mode – apply one color to all cards
3. 🎨 Different Mode – apply different colors to each card
4. 🔁 Toggle between RGB and HEX color formats
5. 📋 Copy generated color to clipboard
6. 📱 Fully responsive (desktop + mobile friendly)
7. ✨ Smooth hover and transition effects

## 🧠 Logic Overview
1. Random colors are generated using Math.random()
2. Mode is controlled using a simple boolean flag
3. Color format (RGB / HEX) is switched dynamically
4. DOM elements update in real time on user interaction
## This project focuses on clarity and simplicity, not over-engineering.

## 🛠️ Tech Stack
1. HTML5 – Structure
2. CSS3 – Layout, responsiveness, UI styling
3. JavaScript (Vanilla) – Logic and interactivity
## No libraries. No frameworks. Pure fundamentals.

## 🎯 Learning Outcomes
1. DOM selection and manipulation
2. Event listeners and UI state handling
3. Responsive design using media queries
4. Writing clean and readable JavaScript
5. UI/UX thinking for small components
