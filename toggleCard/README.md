# 🌗 Theme Toggle (Dark / Light)

# Introduction
A simple and responsive **Theme Toggle UI** built using **HTML, CSS, and Vanilla JavaScript**.
This project allows users to switch between **Dark** and **Light** themes using a dropdown selector.

## 🚀 Features

- 🌗 Switch between **Dark** and **Light** themes
- 🎛️ Dropdown-based theme selection
- ⚡ Instant theme change using class toggling
- 🎨 Clean and minimal UI design
- 📱 Responsive layout for small screens
- 🧩 No external libraries used

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling & responsiveness
- **JavaScript (ES6)** – Theme toggle logic

## ⚙️ How It Works

1. User selects a theme from the dropdown
2. JavaScript listens to the `change` event
3. Based on the selected value:
   - `.dark` class is applied for dark mode
   - `.light` class is applied for light mode
4. Background color updates instantly

## 🧠 Logic Highlights

- Uses `classList.add()` and `classList.remove()`
- Clean event-driven approach
- Simple and readable conditional logic
- Easy to extend for more themes
