## 🚀 30 Days Frontend Foundations

## Introduction
A structured 30-day frontend learning journey focused on building strong logic, solid fundamentals, and real-world UI thinking using HTML, CSS, and JavaScript.

This repository documents my daily practice, problem-solving approach, and progress as a self-learner.

## 🎯 Objective
1. The main goal of this challenge was to:
2. Strengthen JavaScript logical thinking
3. Understand how UI and logic work together
4. Build confidence by solving small but meaningful problems daily
5. Create a solid foundation for Frontend / UI-UX / Full-Stack roles

## This is not a tutorial repo. This is a practice + thinking repo.


## 🧠 What I Focused On (Very Important)

## 1️⃣ Logic Before UI
1. Before styling anything, I focused on:
2. What problem am I solving?
3. What should happen on user interaction?
4. How can I break this into small steps?

## 2️⃣ One Concept at a Time
1. Each day targets one core concept, such as:
2. DOM selection
3. Event handling
4. Conditional logic
5. Loops and arrays
6. State handling using variables

## 3️⃣ Repetition with Variation
1. Same concept → different mini problems
2. This helped in deep understanding, not memorization.

## 📅 Daily Task Pattern
1. Each day follows this structure:
   Small task (not a full project)
   Clear goal (one concept)
   Simple implementation
   Clean code
   Short learnings noted in comments

### 📌 Task & Learning
1. [DAY-01] COLOR GENERATOR
2. [DAY-02] PROFILE CARD UI

## 🛠️ Tech Stack
